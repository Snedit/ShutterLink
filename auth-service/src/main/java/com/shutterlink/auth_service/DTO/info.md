## Info about the DTOs
| DTO | Purpose |
|-----|---------|
| `RegisterRequestDTO` | Register new users |
| `LoginRequestDTO` | Log in users |
| `AuthResponseDTO` | Send tokens + user info |
| `TokenRefreshRequestDTO` | Refresh JWT tokens |
| `TokenValidationResponseDTO` | Validate token for inter-service comms |
| `ChangePasswordDTO` | Change user password |
